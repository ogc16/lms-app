Resources

References, protocols, and glossary for ICS 2203.

Reference Books

  • George Beekman (2008) Tomorrow's Technology and You, 9/E
  • Web technologies & Internet fundamentals textbooks

Key Protocols

  • HTTP – HyperText Transfer Protocol
  • TCP – Transmission Control Protocol
  • IP – Internet Protocol
  • ICMP – Internet Control Message Protocol
  • DHCP – Dynamic Host Configuration Protocol
  • FTP – File Transfer Protocol
  • SMTP – Simple Mail Transfer Protocol

Web Technologies

  • HTML5 – Latest HTML standard
  • XHTML – Stricter HTML syntax
  • CSS3 – Cascading Style Sheets
  • XML – Extensible Markup Language
  • JavaScript – Client-side scripting

DNS Record Types

  • A – IPv4 address
  • AAAA – IPv6 address
  • CNAME – Canonical name (alias)
  • MX – Mail exchange
  • NS – Name server

Glossary

DNS
Domain Name System — translates domain names to IP addresses.
HTML
HyperText Markup Language — standard for creating web pages.
CSS
Cascading Style Sheets — controls presentation and layout.
XML
eXtensible Markup Language — transports and stores data.
TCP/IP
Transmission Control Protocol / Internet Protocol — core Internet protocol suite.
ISP
Internet Service Provider — provides Internet access.
